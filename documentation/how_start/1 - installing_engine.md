## installing engine

first, open your folder of projects and run: 
 ````
  git clone https://github.com/Nicolas-Asafe/tomato-x @tmx
  cd ./@tmx
 ```` 
 install dependencies:
 ````
 npm install
 ````
 build the typescript code:
 ````
 npm run build
 ````
 and run javascript code:
 ````
 npm run start
 ````
if you want to build, and run it in mode watch:
 ````
 npm run init_tx
 ````
 or run it in mode watch:
 ````
 npm run dev_watch
 ````
